* As teclas em todo o jogo precisam ser minúsculas


No menu:
w - opção de cima
s - opção de baixo
f - selecionar

Na tela de password:
esc - retorna para o menu

No jogo:
w - andar para cima
a - andar para a esquerda
s - andar para baixo
d - andar para a direita
f - usar poder
p - tela de password
esc - perde uma vida
